# ISP Open Dataset with GDP from IBGE
Combining two open datasets, data from the public security institute of the state of Rio with GDP data from the municipalities of the state of Rio

# The Open Dataset Used

  - The ISP dataset can be found [here](http://www.ispdados.rj.gov.br/)
  - The IBGE dataset can be found [here](https://www.ibge.gov.br/estatisticas/economicas/contas-nacionais/9088-produto-interno-bruto-dos-municipios.html?=&t=downloads)
